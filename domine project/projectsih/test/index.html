WEB RTC

WebRTC (Web Real-Time Communication) is a free, open-source project that enables real-time communication of audio, video, and data in web browsers and mobile applications without requiring additional plugins or software. It uses JavaScript APIs and peer-to-peer connections to facilitate direct communication between browsers, allowing for applications like video conferencing, online gaming, and file sharing.

IPFS HASH

IPFS (InterPlanetary File System) allows conversion of video files into a unique cryptographic hash, serving as a content identifier. Through IPFS, a video file is broken into chunks and each chunk is assigned a hash, forming a Merkle DAG (Directed Acyclic Graph). The resulting hash represents the entire video, enabling decentralized storage and retrieval without reliance on traditional servers.

BLOCK CHAIN

Storing IPFS hash links in a blockchain provides a secure and immutable record of the content's existence and ownership. By anchoring the hash in the blockchain, it becomes resistant to tampering or deletion, enhancing trust and verifiability. This approach ensures decentralized, permanent storage of data references, with blockchain acting as a decentralized ledger for tracking content integrity and ownership.
